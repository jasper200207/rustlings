// Make the code print a greeting to the world.

# Answer

```rust
// intro2.rs
// Make the code print a greeting to the world.
// Execute `rustlings hint intro2` or use the `hint` watch subcommand for a hint.

fn main() {
    println!("Hello {}!", "world");
}
```

# Solution

rust에서 출력을 포멧하려면 `{}`와 `,`를 사용한다.
